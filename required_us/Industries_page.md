1. Industries Page Overview
User Story 1

As a visitor
I want to see all industries the company serves
So that I can quickly identify if my business is covered

Acceptance Criteria
Given the visitor opens the Industries page
When the page loads
Then the page should display a list of industries served
And each industry should be clearly labeled
User Story 2

As a business client
I want a structured layout of industries
So that I can easily navigate and scan relevant sectors

Acceptance Criteria
Given the industries page is displayed
When the visitor views the content
Then industries should be presented in a grid or list format
And each item should be visually distinct
2. Individual Industry Cards
User Story 3

As a visitor
I want to see a brief description of each industry
So that I can understand how services apply to my sector

Acceptance Criteria
Given the industry cards are displayed
When the visitor views a card
Then it should include industry name
And a short description
And a relevant icon or image
User Story 4

As a user
I want each industry card to be clickable
So that I can explore more details

Acceptance Criteria
Given the visitor clicks an industry card
When the action is triggered
Then the system should navigate to a detailed section or page for that industry
3. Industry Detail Section
User Story 5

As a business client
I want to see how services are tailored to my industry
So that I can evaluate relevance

Acceptance Criteria
Given the visitor opens an industry detail
When the section loads
Then it should describe how security, housekeeping, and facility services apply to that industry
And include practical use cases
User Story 6

As a decision-maker
I want to understand specific challenges addressed
So that I can relate to my business problems

Acceptance Criteria
Given the industry detail section is visible
When the visitor reads it
Then it should highlight common challenges and solutions for that industry

6. Call-to-Action (CTA)
User Story 9

As a visitor
I want to inquire directly from the industry section
So that I can request services tailored to my sector

Acceptance Criteria
Given the visitor is viewing an industry section
When the section is displayed
Then a "Get Quote" or "Contact Us" button should be visible
And clicking it should navigate to the inquiry form
User Story 10

I want clear CTAs after reading content
So that I know what to do next

Acceptance Criteria
Given the visitor reaches the end of an industry section
When the content ends
Then a strong CTA should be displayed

Acceptance Criteria
